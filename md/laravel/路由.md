###### 路由参数
* 必填参数 不传参数找不到   

```
Route::get('user/{id}', function ($id) {
    return 'User '.$id;//不传参数404 
});

多个参数,控制器或者回调顺序对即可
路由的参数{} 内，并且参数名只能为字母，同时路由参数不能包含 - 符号
Route::get('posts/{post}/comments/{comment}', function ($postId, $commentId) {
   
   
});


```

* 可选参数 {name?} 可选路由参数,但是回调必须有默认值
